Features

PHP statistic

Review of requests time, memory peak usage, site load, error pages.

Slow and heavy pages

Analyse log of slowest (request processing time) and heavest (memory usage) pages.

Error pages

Analyse pages with error HTTP statuses.

Live mode

View in realtime information about request processing.
